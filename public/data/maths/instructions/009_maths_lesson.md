# Screen 2: Variables - The Building Blocks

A **variable** is a letter that represents a number. It's called a variable because its value can vary (change).

## Common Variables

**We usually use these letters:**
- **x** and **y** - Most common in algebra
- **a**, **b**, **c** - For constants or unknowns
- **n** - Often used for counting (1st, 2nd, nth)
- **t** - For time
- **d** - For distance
- **Any letter works!** - Even z, p, q, m, etc.

## What Variables Represent

**Example 1: Unknown Values**
```
x = the number I'm thinking of
If I say "I'm thinking of x"
and x = 7
then I'm thinking of 7
```

**Example 2: Changing Values**
```
n = your age
When you're 14: n = 14
Next year: n = 15
The letter stays the same, the value changes!
```

**Example 3: General Rules**
```
Perimeter of square = 4 Ã— side length
Using algebra: P = 4s

This works whether s = 5cm, s = 10cm, or any size!
```

## Important Points About Variables

**1. A variable represents ONE number at a time**
```
If x = 5, then x = 5 everywhere in that problem
You can't have x = 5 AND x = 7 in the same problem!
```

**2. Different letters can have different values**
```
x = 3 and y = 7 is fine
They're different variables
```

**3. The same letter means the same value**
```
In x + x + x
All three x's are the same number
If x = 4, then x + x + x = 4 + 4 + 4 = 12
```

# Screen 4: Expressions - Putting It Together

An **expression** is a collection of numbers, variables, and operations. Think of it like a phrase in maths.

## What is an Expression?

An expression is **NOT an equation** - it doesn't have an equals sign!

**These are expressions:**
- 3x + 5
- 2a - 7
- 4xÂ² + 3x - 2
- x/2 + y

**These are equations (NOT expressions):**
- 3x + 5 = 11 (has equals sign)
- y - 3 = 10 (has equals sign)

## Parts of an Expression

**Each part separated by + or - is called a TERM**

## Terms Explained

A **term** is a part of an expression:

**Example: 4x - 2y + 7**
```
This expression has 3 terms:
Term 1: 4x
Term 2: -2y  (the minus sign stays with it!)
Term 3: +7   (also written as just 7)
```

**Types of Terms:**

**Variable Terms** - contain letters:
- 3x, -5y, 7a, xy

**Number Terms (Constants)** - just numbers:
- 5, -3, 10, 100

## Coefficients

The **coefficient** is the number in front of a variable:

**Examples:**
- In **5x**, the coefficient is 5
- In **3a**, the coefficient is 3
- In **x**, the coefficient is 1 (invisible!)
- In **-2y**, the coefficient is -2

**Remember:** If there's no number visible, the coefficient is 1!
